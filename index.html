自己紹介

バックグラウンド・興味・現在やっていること

中国・黒竜江省出身、2002 年生まれです。中学卒業後の 2017 年 9 月に来日し、ゼロから日本語を学びました。 相模原市の学校に通い、2019 年に大清水中学校を卒業、 その後相模原弥栄高等学校を経て 2022 年に東京都市大学へ入学しました。

現在は計算機ソフトウェア研究室に所属し、 企業との共同研究として SBOM 生成ツールの精度評価に取り組んでいます。 約 1500 件の GitHub プロジェクトを対象に 4 種のツールを比較し、 精度に影響する要因の分析を行っています。

来日当初にお世話になった日本語教室でボランティアとしても活動しており、 ローテーション管理の自動化 Web システムを開発・導入しました。

「静かな青、積み重ねる歩み」をモットーに、落ち着いて誠実に取り組むことを大切にしています。 呼び名は「はい」でも「ペイ」でも大丈夫です。

Projects

研究・プロジェクト

何をやったか、担当、結果

研究（企業共同） 進行中

SBOM 生成ツールの精度評価・要因分析

ソフトウェアサプライチェーンセキュリティの向上を目的に、 4 種の SBOM 生成ツールを対象として精度を比較・分析。 再現率・適合率の指標で評価し、精度に影響する要因を明らかにした。

SBOM Python 脆弱性管理 サプライチェーン 企業共同研究

担当：実験設計・処理自動化・比較分析   結果：約 1500 件を処理、精度に影響する 4 要因を特定

開発（ボランティア） 完成・運用中

日本語教室 ローテーション自動化 Web システム

ボランティアとして関わる日本語教室で、毎月の担当割り当てを 手作業で行う非効率を解消するため Web システムを設計・開発。 講師ごとの出席状況・個別事情など複雑な条件を「必須条件」と 「考慮条件」に分類し、自動スケジューリングを実装した。

Web 開発 スケジューリング スマホ対応 UI DX

担当：要件定義・設計・開発   結果：割り当て作業を数時間 → 数分に短縮

グループ活動（SDPBL） 完成

学内食堂 混雑可視化シミュレーター

学科横断のグループで学内食堂の混雑問題に取り組んだ。 議論の進行役として意見を整理し方向性をまとめる役割を担い、 提案内容を直感的に確認できるシミュレーションプログラムを実装した。

シミュレーション チーム開発 ファシリテーション

担当：議論進行・シミュレーター実装   結果：全員合意の解決策プロトタイプを完成
